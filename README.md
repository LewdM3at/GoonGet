# GoonGet

CLI tool for fetching Rule34, Gelbooru content directly in your terminal.

## 📌 Overview

GoonGet is a command‑line utility that fetches images and GIFs from Rule34 and displays them directly in your terminal using chafa. 

## ✨ Features

- Fetch images and GIFs from Rule34 and Gelbooru

- Display media directly in your terminal via chafa

- Search by tags

- Slideshow mode

- Settings available for "default" tags that you always want included/excluded

## 📦 Installation
Arch Linux (AUR)

    yay -S goonget
